Parameters


Description
General parameters of a FragScape project must be set before any processing.

Parameters
Workspace must be set to produce FragScape output layers. This directory is used to build relative paths.
If option is checked, intermediate layers are stored in Workspacee/tmp directory.
Projection must be a metric projected coordinate system. It defines shape are area of features so that it must be choosed according to geographic extent and project specific criterias (for instance : does shape matter ?).

Results
Current parameters value can be checked in visualisation table.
