Parameters


Description
General parameters of FragScape project must be set before any processing.

Parameters
Workspace must be set to produce FragScape output layers. This directory is used to build relative paths.
Mode defines data destination format. Depending on land cover data, either Vector (more precise if input data is already vector) either Raster (faster but limited by available RAM) must be chosen. See documentation for benchmark.
Resolution defines pixel size in georeferenced unit (meter). Available and mandatory only in Raster mode.
Extent layer can be specified so that every input data is clipped to such extent. Can be vector or raster layer.
Projection must be a metric projected coordinate system. It defines shape are area of features so that it must be choosed according to geographic extent and project specific criterias (for instance : does shape matter ?).
Press Save intermediate layer to keep track of processing results in directory Workspace/tmp.

Results
Current parameters value can be checked in visualisation table.
