Paramètres


Description
Les paramètres généraux du projet BioDispersal doivent être renseignés avant tout traitement.

Paramètres
Le dossier de travail doit être défini pour générer les couches de sortie de FragScape. Il sert de dossier de référence pour construire les chemins relatifs.
Le mode définit le format de données utilisé pour le calcul des indicateurs. Le mode Vecteur est plus précis si l'occupation du sol est déjà au format vecteur. Le mode Raster est plus rapide (bien que limité par la mémoire vive disponible). La documentation contient des exemples de temps de calcul.
La Résolution définit la taille d'un pixel en unité géoréférencées (en m√®tres). Disponible et obligatoire seulement en mode Raster.
La Couche d'emprise peut être renseignée afin de découper les données d'entrée selon la même emprise.
Le système de coordonnées doît être une projection. Il détermine l'aire et la forme des polygones et a donc une influence directe sur les résultats. Il est choisi en fonction de l'emprise des données et de critères spécifiques au projet tels que la précision du calcul des aires (et donc des indicateurs) ou la précision des formes géométriques (prise en compte d'un indice de compacité par exemple).
