Paramètres


Description
Les paramètres généraux du projet FragScape doivent être renseignés avant tout traitement.

Paramètres
Le dossier de travail doit être défini pour générer les couches de sortie de FragScape. Il sert de dossier de référence pour construire les chemins relatifs.
Le Mode définit le format de données utilisé pour le calcul des indicateurs. Le mode Vecteur est plus précis si l'occupation du sol est déjà au format vecteur. Le mode Raster est plus rapide (bien que limité par la mémoire vive disponible). La documentation contient des exemples de temps de calcul.
La Résolution définit la taille d'un pixel en unité géoréférencées (en mètres). Disponible et obligatoire seulement en mode Raster.
La Couche d'emprise peut être renseignée afin de découper les données d'entrée selon la même emprise.
Le système de coordonnées doît être une Projection métrique (Lambert-93 par défaut). Il doit être choisi en fonction de l'emprise gêographique et des enjeux spˆcifique du projet.
