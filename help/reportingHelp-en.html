Results


Description
This final step computes fragmentation metrics.

Parameters
Input layer is automatically updated when step 3 is performed and can be specified by user.
Reporting layer contains territory units for which fragmentation metrics are computed (e.g. metrics are computed for each feature of reporting layer). Cross-Boundary Connection (CBC) is an update of effective mesh size to mitigate "border effect" (natural patches clipped at administrative borders of the territory). See Jaeger article for further details.
Specify Unit of area for results metrics (by default results are rounded to 5 digits precision).

Results
Territory global effective mesh size is displayed in user interface.
Output layer with fragmentation metrics for each reporting unit is loaded in QGIS with an automatic graduated style according to effective mesh size value (from red for fragmented territories to green for less fragmented territories).
