Results


Description
This final step computes fragmentation metrics.

Parameters
Input layer is automatically updated when step 3 is performed.
Input features can be filtered, for instance by specifying a minimal area.
Reporting layer contains territory units for which fragmentation metrics are computed (e.g. metrics are computed for each feature of reporting layer). CUT method clips any input feature according to reporting feature extent. CBC method weights such intersected area by initial area of the input feature. To apply CBC method, input features that intersect reporting units must not be clipped (nor intersect input layer boundaries).

Results
Territory global effective mesh size is displayed in user interface (step 4).
Output layer contains fragmentation metrics for each reporting unit.
