Occupation du sol


Description
Cette étape permet de sélectionner les éléments non fragmentants (milieux naturels et semi-naturels) à partir d'une couche d'occupation du sol.

Paramètres
La Couche d'entrée (occupation du sol) peut être au format raster ou vecteur et doit couvrir l'intégralité du territoire. Les données sont découpées selon la couche d'emprise si elle a été renseignée dans l'onglet Paramètres.
Le Mode de sélection Par valeur de champ permet d'extraire automatiquement les postes d'occupation du sol (valeurs du Champ de sélection renseigné). Le champ de description des types d'occupation du sol peut être séléctionné s'il existe mais est optionnel.
Appuyer sur le bouton pour Afficher les valeurs de champ et sélectionner les éléments non fragmentants (milieux naturels et semi-naturels).
Si les critères de sélection sont plus complexes (dépendant de la géométrie ou de plusieurs champs combinés), il faut alors utiliser le mode de sélection Par expression.

Résultat
Appuyer sur le bouton Lancer la sélection pour sélectionner et agréger les classes cochées.
En mode vecteur, les entités sont sélectionnées et leur géométrie regroupée (le regroupement de géométrie peut être très long en fonction de la précision et l'emprise des données). En mode raster, la couche est reclassifiée et normalisée d'après les paramètres reseignées (résolution/emprise).
Le couche résultat de cette étape est chargée dans QGIS.
