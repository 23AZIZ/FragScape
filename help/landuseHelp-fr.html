Occupation du sol


Description
Cette étape permet de sélectionner les éléments non fragmentants à partir d'une couche d'occupation du sol (par exemple les espaces naturels).

Paramètres
La couche d'occupation du sol doit couvrir l'intégralité du territoire mais il peut s'avérer nécessaire d'utiliser une couche couvrant un plus large territoire si la méthode CBC (cf étape 4) est utilisée afin de limiter l'effet frontière.
Il est possible de couper les données d'entrée pour diminuer les temps de traitement.
Le mode de sélection Par valeur de champ permet d'extraire automatiquement les postes d'occupation du sol depuis le champ renseigné puis de cocher les éléments non fragmentants. Le champ de description correspond à un champ de description des types d'occupation du sol s'il existe mais est optionnel.
Si les critères de sélection sont plus complexes (dépendant de la géométrie ou de plusieurs champs combinés), il faut alors utiliser le mode de sélection Par expression.

Résultat
Lancer la sélection en appuyant sur le bouton correspondant. Les éléments non fragmentants sont alors sélectionnés et regroupés (cette étape peut être longue pour de larges territoires). Le résultat est chargé dans QGIS et sauvegardé dans la couche DossierDeTravail/outputs/landuseSelectionDissolve.gpkg.
