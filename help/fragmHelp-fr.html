Fragmentation


Description
Cette étape permet d'intégrer les éléments fragmentants à la couche produite en étape 2.
Pour chaque type d'élément ciblé, il faut renseigner tout ou partie des paramètres puis Enregistrer la sélection. Le paramétrage retenu apparaît alors comme une nouvelle ligne dans la table de visualisation.

Paramètres
La couche d'entrée contenant les éléments fragmentants peut être ponctuelle, linéaire ou surfaçique. Il est possible de découper cette couche comme en étape 2.
Expression : critère de sélection des entités depuis la couche d'entrée. Si l'expression n'est pas renseignée, toutes les entités sont sélectionnées.
Tampon : application d'une zone tampon autour des entités sélectionnées pour modéliser leur emprise au sol. Obligatoire pour les données ponctuelles et linéaiares.
Identifiant: identifiant unique du paramétrage courant

Résultat
Pour chaque paramétrage, les traitements associés (découpage, sélection, application de zone tampon) sont effectués et stockés dans une couche correspondant à l'identifiant.
Toutes ces couches sont alors fusionnées, une différence géométrique est effectuée avec le résultat de l'étape 2 et les polygones résultants sont transformés en géométrie simple. Le résultat de ces traitements est sauvegardé dans la couche DossierDeTravail/outputs/landuseFragmSingleGeom.gpkg.
