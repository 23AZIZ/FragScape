Fragmentation


Cette étape est optionnelle

Description
Cette étape est optionnelle
Cette étape permet d'intégrer des données complémentaires aux classes d'occupation du sol sélectionnées.

Paramètres
Choisir une Couche d'entrée au format vecteur ou raster.
Si la couche d'entrée est au format vecteur, il est possible de sélectionner les entités vérifiant l'Expression renseignée et de leur appliquer une zone Tampon pour modéliser leur emprise au sol. Si l'expression est vide, toutes les entités sont sélectionnées.
Choisir un Identifiant unique pour le paramétrage courant.
Spécifier si les données sélectionnées contribuent à la Fragmentation ou si elles sont à rajouter aux élements naturels.
Appuyer sur Enregistrer la sélection pour ajouter le paramétrage courant et le hiérarchiser (nouvelle ligne du tableau) en fonction de sa priorité.

Résultat
Appuyer sur Intégrer les données complémentaires.
Pour chaque paramétrage, les traitements associés (découpage, sélection, application de zone tampon) sont effectués et stockés dans une couche temporaire.
Ces couches sont alors fusionnées avec la couche d'occupation du sol (sortie de l'étape 2) dans l'ordre spécifié (hiérarchisation). La couche résultante est chargée dans QGIS.
